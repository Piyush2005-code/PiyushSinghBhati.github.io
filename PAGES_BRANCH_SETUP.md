# GitHub Pages Setup - Pages Branch Configuration

## Current Setup

Your repository is organized to keep `main` branch clean:

- **`main`** branch - Source code (protected, no auto-deployment)
- **`pages`** branch - GitHub Pages deployment source (with CNAME configuration)
- **`develop`** branch - Development work
- **`gh-pages`** branch - Auto-generated by GitHub Actions (build output)

## How It Works

1. You make changes on `pages` branch
2. Push to `pages` branch on GitHub
3. GitHub Actions automatically:
   - Builds your React app
   - Runs tests and linting
   - Deploys to `gh-pages` branch
4. GitHub Pages serves from `gh-pages` branch at `piyushsinghbhati.com`

## Setup Steps

### Step 1: Push Your Code

```bash
# Make sure you're on pages branch
git checkout pages

# Commit any pending changes (everything should be clean)
git status

# Push pages branch to GitHub
git push origin pages
```

### Step 2: Configure GitHub Repository Settings

1. **Go to GitHub** → Your repository
2. **Settings** → **Pages**
3. Configure as follows:

   | Setting | Value |
   |---------|-------|
   | Source | Deploy from a branch |
   | Branch | `gh-pages` (automatically created by workflow) |
   | Directory | / (root) |

4. Under "Custom domain":
   - Enter: `piyushsinghbhati.com`
   - Click Save

5. **Check "Enforce HTTPS"** (wait ~5 minutes for certificate)

### Step 3: Configure DNS at Namecheap

(See [DOMAIN_SETUP.md](DOMAIN_SETUP.md) for detailed instructions)

Add these A records to Namecheap Advanced DNS:
```
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```

### Step 4: Monitor Deployment

1. Go to **Actions** tab on GitHub
2. Watch the CI/CD Pipeline workflow run
3. Once complete, your site will be deployed to `gh-pages` branch
4. After DNS propagates (24-48 hours), visit `https://piyushsinghbhati.com`

## Troubleshooting

### Workflow not running
- Verify you pushed to `pages` branch (not main)
- Check GitHub Actions tab for warnings
- Verify branch protection rules aren't blocking

### CSS/JS not loading
- Verify `public/CNAME` file exists
- Check that build completed successfully
- Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)

### Domain shows 404
- Verify DNS A records are correct on Namecheap
- Wait 24-48 hours for DNS propagation
- Test with: `dig piyushsinghbhati.com`

### HTTPS certificate error
- Wait 5-10 minutes after enabling "Enforce HTTPS"
- Certificate is being provisioned by Let's Encrypt
